
✅ Resume Screening System – Project Description

The Resume Screening System is a web-based application developed using Python and Streamlit that automates the process of filtering and shortlisting resumes. The system allows users to upload multiple resumes in PDF or DOCX format, extracts text using libraries like PyPDF2 and python-docx, and processes the data using NLP techniques.

It uses machine learning algorithms from Scikit-learn to analyze resumes and match them with job descriptions based on skill similarity, keywords, and relevance score. The system ranks candidates and displays results in an easy-to-understand format, helping recruiters make faster and more accurate hiring decisions.

🚀 Key Features

📄 Upload multiple resumes (PDF/DOCX)

🔍 Automatic text extraction and preprocessing

🧠 NLP-based skill and keyword matching

📊 Resume ranking based on job description

⚡ Real-time results with an interactive UI

📈 Basic data visualization (optional)

📂 Supports structured candidate comparison
